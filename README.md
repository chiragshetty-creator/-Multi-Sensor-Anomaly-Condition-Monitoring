# -Multi-Sensor-Anomaly-Condition-Monitoring
📌 Project Overview

This project focuses on detecting abnormal behavior in industrial machines using multi-sensor data such as Temperature, Vibration, Pressure, Energy Consumption, and Production Units.

The system identifies anomalies using statistical methods and machine learning techniques, and visualizes results through interactive dashboards.

⸻

🎯 Objective • Detect abnormal sensor readings in industrial machines • Reduce machine failures and downtime • Provide insights using data visualization

⸻

🏭 Problem Statement

Industrial machines generate large volumes of sensor data. Identifying anomalies manually is difficult and time-consuming.

This project automates anomaly detection and condition monitoring using data analytics and AI techniques.

⸻

🛠️ Tools & Technologies • Python (Pandas, NumPy, Scikit-learn) • SQL (CTE, Aggregations, CASE WHEN) • Power BI (Dashboard & Visualization) • MS Excel (Data Cleaning & Analysis) • Statistical Methods (Mean, Standard Deviation, Z-score)

⸻

⚙️ Methodology

Data Preprocessing • Handling missing values • Data formatting and cleaning • Timestamp alignment

Statistical Analysis • Mean ± 3σ anomaly detection • Z-score normalization

SQL-Based Analysis • Used CTEs to calculate mean and standard deviation • Applied threshold logic to detect anomalies

Machine Learning • Implemented Isolation Forest algorithm • Detected top anomalous machine readings

Condition Index • Created using normalized sensor values • Ranked machines based on health condition

Dashboard Development • Built interactive dashboards using Power BI • Included anomaly trends, KPIs, and slicers

⸻

📊 Key Features • Multi-sensor anomaly detection • Machine-wise condition monitoring • Severity classification (Low / Medium / High) • Automated reporting using Python • Interactive Power BI dashboards
